# Who-is-suspiciousとは

Among Us MODの「Who-is-suspicious」略して「wis」
直訳すると、「疑わしいのは誰？」となります。<br>
開発者が少なく、知識不足のため更新が遅くなる場合があります。
ポンコツですが皆さんが楽しく遊べるようにします！

## 現在の機能
- 1人開始
- wisスタンプ
- 公開ルームにできなくなった

## 必要
- Steam版のアモングアス
